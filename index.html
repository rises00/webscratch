How to Create a GitHub Repository

Follow these simple steps to create a GitHub repository:

  1. Log in to your GitHub account.
  2. Click on the + (plus icon) in the top-right corner of the page and select New repository.
  3. Fill in the repository details:
    • Repository name: Choose a unique name for your repository.
    • Description (optional): Provide a short description of your repository.
    • Visibility: Select Public or Private depending on your preference.
  4. Click the Create repository button to finalize.
